# General description about what each script does.
#
GE_clean.py -> To delete files from certain simulation. Edit manually parentDir

GE_galaxy_photometry_simulator.py -> Script to make a simulation.

GE_launch_complete_simulation.py -> Script that launches full simulation.

GE_launch_simulation_set.py -> Script that calls GE_galaxy_photometry_simulator.py using parameters file.

GE_recolect.py -> Collects results from certain simulation, check them and save to a file.
