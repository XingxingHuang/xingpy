# General description about folders
#
Catalogs -> Individual catalogs from simulations

CondorFiles -> Condor configuration files

Entornos -> .tar.gz files containing needed files for each simulation

Initial -> From where simulations are launched

LockFiles -> .lock and .done files for simulation status

ParamFiles -> Parameters files that define each simulation

Results -> Final catalogs, plots, ...
